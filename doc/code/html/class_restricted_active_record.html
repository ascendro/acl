RestrictedActiveRecord Class File This class serves as a base-class for all the objects which have to control their access.

Author:
dispy <dispyfree@googlemail.com> LGPLv2 This class is intended as a base for objects which have restrictions on their access It automatically checks, if the current user has the permissions to commit the regular CRUD-tasks
